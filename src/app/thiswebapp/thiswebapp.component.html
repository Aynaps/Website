Objective

To create my own personal website to get web development experience/full stack, and something easy for employers to navigate around instead of boring resume

How it works

The framework I used for this website is AngularJS AngularJS is a javascript framework that uses components and all components run off of the main AppComponent each component generate html in another component or host its own webpage, just like this one!

Functionality

Given how youve already got here, you know that clicking on one of the cards under 'My Projects' takes you to the page I made for that project You can either click the back arrow to goto the homepage or you can click the 'Welcome' in the top left corner. The github logo in the top right hosts my projects with the source code and will open my repos under a new tab. Also if you press the Jam Button above Pepe the Frog he will start dancing to Daft Punk - One More Time
